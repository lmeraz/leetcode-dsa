## graphs-01
Build a graph
```python
from collections import defaultdict
def build_graph(edges):
    graph = defaultdict(list)
    for x, y in edges:
        graph[x].append(y)
        # graph[y].append(x)
        # uncomment the above line if the graph is undirected
    return graph
```
